# ANTIGRAVITY.md - Omikuji App Project Constitution

## 1. プロジェクト概要
- **名称**: Cyber Omikuji 2077 (サイバーおみくじ)
- **目的**: Webブラウザで動作する、近未来的なネオン・サイバーパンク風おみくじアプリの作成。
- **ターゲット**: 刺激的なビジュアル体験を求めるユーザー。未来的・SF的な世界観が好きな人。

## 2. 技術スタック
- **Core**: HTML5, JavaScript (Vanilla)
- **Styling**: CSS3 (Vanilla) - ネオンカラー（ピンク、シアン、紫）とダークモード。グリッチエフェクトや発光表現を使用。
- **Framework**: なし (No Framework)

## 3. MVP (Minimum Viable Product) 要件
- **機能**:
    - 「CONNECT TO SHRINE (おみくじを引く)」ボタンを設置。
    - ボタンクリックで「DAI-KICHI (大吉)」「MEGA-KYO (大凶)」などの結果をランダム表示。
- **デザイン**:
    - **Cyberpunk Style**: 黒背景にネオン管のような発光デザイン。
    - **UI**: 透過ガラス風パネル、走査線ライン、LEDインジケーター。
    - **Font**: 近未来的な等幅フォントやGoogle Fontsの「Orbitron」などを採用。
- **演出 (Vibe Check)**:
    - **大吉**: 画面全体がゴールド/レインボーに輝き、祝福のパーティクルエフェクト。
    - **大凶**: 画面全体が赤く点滅し、システム警告（WARNING）のようなグリッチノイズ演出。
    - 通常時もホバーエフェクトなどでインタラクティブな動きをつける。
